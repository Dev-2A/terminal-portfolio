# 💻 Terminal Portfolio

브라우저에서 터미널처럼 명령어로 탐색하는 인터랙티브 개발자 포트폴리오입니다.

> "전직 사서 → AI/ML 엔지니어" 내러티브를 터미널 UX로 전달합니다.

## 🎯 주요 기능

- `help` — 사용 가능한 명령어 목록
- `whoami` — 개발자 소개
- `ls projects` — 22개 토이 프로젝트 탐색
- `cat about.md` — 상세 자기소개
- `open <project>` — 프로젝트 GitHub/배포 페이지 열기
- `theme <name>` — 터미널 테마 변경

## 🛠️ 기술 스택

- React + Vite
- Tailwind CSS
- GitHub Pages
